     Zeylab HRMS Demo - Quick Start Guide
===============================================

FASTEST WAY TO RUN:

Method 1 - Standalone Version (Recommended):
1. Double-click: START-DEMO.bat
2. Or type: node ZeylabHRMS-Standalone.js

Method 2 - Extract Demo Package:
1. Extract: Zeylab-HRMS-Demo-Final.tar.gz
2. Enter folder: zeylab-hrms-demo
3. Double-click: START-DEMO.bat
4. Or type: node server.js
